### XP Bars - Gradient Fill with Shimmer

```css
/* Gradient XP Bar */
.xp-bar {
  background: linear-gradient(
    to right,
    var(--cyberpunk-blue),
    var(--cyberpunk-neon),
    var(--cyberpunk-pink)
  );
  animation: shimmer 2s infinite;

/* Shimmer Effect */
@keyframes shimmer {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(200%);
  }
}

.shimmer-overlay {
  background: linear-gradient(
    to right,
    transparent,
    rgba(255, 255, 255, 0.2),
    transparent
  );
}
```
